# Docker & Others

A collection of Docker, Kubernetes, and infrastructure examples for learning and lab purposes.

## Contents

| Directory | Description |
|---|---|
| `app/` | Sample Python API application |
| `dockerfile/` | Dockerfile examples and variants |
| `dockerfile_example/` | Dockerfile patterns (multistage, entrypoint, webserver) |
| `elastic/` | Elasticsearch + Kibana stack with Docker Compose |
| `kubernetes/` | Kubernetes setup files and join instructions |
| `n8n/` | n8n workflow automation with Docker Compose |
| `redis/` | Redis and Sentinel configuration |
| `scripts/` | Helper scripts (install, start, stop server) |
| `zookeeper/` | Zookeeper Docker Compose setup |
| `Site_Lab/` | Lab website |
| `training2019/` | Training materials from 2019 |

## Requirements

- Docker
- Docker Compose
- Kubernetes (for k8s examples)

## Usage

Each directory contains its own configuration. Navigate to the desired directory and follow the specific setup instructions.
