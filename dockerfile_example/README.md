# Dockerfile Examples & References

Reference examples and descriptions for common Dockerfile instructions.

## Files

- `options_description.txt` — Description of Dockerfile instructions (ADD, CMD, COPY, ENTRYPOINT, ENV, EXPOSE, FROM, RUN, USER, VOLUME, WORKDIR)
- `go_multistage.txt` — Multi-stage build example with Go
- `webserver_simple.txt` — Simple webserver Dockerfile
- `webserver_entrypoint.txt` — Webserver with custom ENTRYPOINT
- `entry_figlet` — Entrypoint example using figlet
- `hello.c` / `hello` — C hello world binary for container examples
