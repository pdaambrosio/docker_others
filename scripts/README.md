# Scripts

Helper shell scripts for managing an Apache HTTPD server.

## Scripts

| Script | Description |
|---|---|
| `install_dependencies` | Installs Apache HTTPD via yum |
| `start_server` | Starts the httpd service |
| `stop_server` | Stops the httpd service if running |

## Usage

```bash
bash install_dependencies
bash start_server
bash stop_server
```
